Welcome to Git lesson

This is the git lesson for batch 16.

Step by step guide:

  1. Install Git from git-scm.com
  2. Configure Git name: git config --global user.name "Your Name"
  3. Configure Git email: git config --global user.email "youremail@example.com"
  4. Initialize git: git init
  5. Merge commits: git merge branchname
  6. Create Branch: git branch branchname
