Welcome to Git lesson

This is the git lesson for batch 16.

Step by step guide:

  1. Install Git from git-scm.com
  2. Configure Git: git config --global user.name "Your Name"
  3. Set your email: git config --global user.email "you@example.com"
  4. Initialize a repository: git init
  5. Create a file and add it: git add filename
  6. Commit changes: git commit -m "Initial commit"
  7. Check status: git status
  8. View history: git log
  9. See changes: git diff
  10. See branches: git branch
  11. Branching: git switch -c branchname
  12. Delete branch: git branch -d branchname
  13. Undo changes in WD: git restore filename
  14. We start to use Git Hub
  15. For quickly commiting changes to already tracked files: git commit -am "Initial commit
  16. Commit with VIM: git commit
  17. Go to web
